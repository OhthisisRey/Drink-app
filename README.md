# Shaker 🍸

A minimalist home cocktail finder that suggests drinks based on the ingredients available in your bar.

---

## Live Demo

[Open the App](https://drink-app.vercel.app)

---

## Features

- Ingredient-based cocktail discovery
- Shows drinks you can make right now
- Shows drinks missing only one ingredient
- Step-by-step cocktail instructions
- Favorites system
- Random “Tonight’s Pick” cocktail
- Clean dark UI
